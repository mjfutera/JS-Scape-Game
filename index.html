About

Cast & Collect is a browser-based 3D raycasting maze game rendered entirely on an HTML5 Canvas using a custom DDA (Digital Differential Analyzer) algorithm — no WebGL required.

Explore procedurally generated mazes and collect all the gems!

Controls

  • W / A / S / D — Move
  • Mouse — Look around
  • Arrow Keys — Alternative turn/move
  • ESC — Pause / release mouse
